# 🎨 BootStrap Creative Design - Custom CSS Library

This project is part of a task to **build a personal CSS library** and use it to recreate a website design inspired by the **Creative template from Start Bootstrap**.

Instead of using Bootstrap itself, I created my own reusable **CSS utility library** and used it to construct the full layout.

# 🎨 CSS Library

## base.css

This file acts as a **custom CSS utility framework** and contains reusable classes such as:

### Layout
- container
- row
- flex utilities
- alignment utilities

### Spacing
- padding utilities
- margin utilities
- gap utilities

### Typography
- font sizes
- font weights
- text colors

### Components
- buttons
- inputs
- links

### Helpers
- width helpers
- position helpers
- display utilities
- transitions

This allows building the layout **quickly using reusable classes**.
